# 🛡️ Nexus Shield AI Frontend

The frontend interface for **Nexus Shield AI**, an AI-powered website security analysis platform.

The frontend provides the user experience for scanning websites, viewing security results, managing accounts, and receiving security insights.

🌐 **Live Website:** https://shield.zenithish.com

---

## ✨ Features

### 🔍 Security Dashboard

* Scan websites for security issues
* View security scores
* Review detected vulnerabilities
* Understand security improvements

### 🧠 AI-Powered Insights

* View AI-generated explanations
* Understand technical security findings
* Get clearer recommendations

### 👤 User Accounts

* Create an account
* Secure login system
* Personalized dashboard experience
* Manage security preferences

### 📊 Scan History

* View previous website scans
* Review past security results
* Manage notification settings for scans

### 📧 Email Features

* Send test security emails
* Enable or disable email alerts
* Receive weekly security summaries

---

## 🏗️ Technology Stack

* **React** — Frontend framework
* **Vite** — Build tool and development environment
* **JavaScript** — Application logic
* **CSS** — User interface styling
